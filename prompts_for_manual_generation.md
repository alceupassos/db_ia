# Prompts para Geração de Infográficos (Gemini 3.0 Nano Banana Pro)

Se a geração automática falhar devido a limites de API, use estes prompts diretamente na interface do Gemini.

> **Dica:** Copie e cole o prompt exato. Se o resultado não for um SVG, adicione: "Please output valid SVG code only".

## 1. Acesso aos Templates (`step-1-access.svg`)

**Prompt:**

```text
Create a professional SVG infographic showing a user navigating to a legal templates section. Show a sidebar navigation menu on the left with an arrow pointing to a templates folder/icon on the right. Use a dark theme with purple (#8B5CF6, #A78BFA) and dark backgrounds (#0D0A14, #1a1625). Include minimal, clean design with modern UI elements. Output ONLY the raw SVG code, no markdown, no explanations.
```

## 2. Categorias (`step-2-category.svg`)

**Prompt:**

```text
Create a professional SVG infographic showing different legal document categories organized as folders. Display folders for: Contracts, Powers of Attorney, NDAs, MOUs, Patents, Bids. Use a grid layout with purple accents (#8B5CF6, #A78BFA) and dark background (#0D0A14). Modern, clean design. Output ONLY the raw SVG code.
```

## 3. Seleção de Empresa (`step-3-company.svg`)
